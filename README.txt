In deze opdracht is het de bedoeling om een JSON bestand om te zetten naar een CSV bestand.
Hiervoor is een virtualenvironment gemaakt. Dit is net een Docker container maar dan in Python zelf.
De reden dat ik voor Python heb gekozen is, omdat er heel veel libaries zijn binnen Python die het 
makkelijk maken om zulke scripts te schrijven.

In dit project heb je ook een requirements.txt bestand staan. Hierin kun je alle libraries zien die zijn gebruikt voor deze opdracht.
Het enige wat je moet doen is het starten van de environment en ervoor zorgen dat alle libraries middels de requirements.txt is geinstalleerd.
